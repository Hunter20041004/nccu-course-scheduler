# AI 功能共用個人資料設計規格

## 目標

讓「AI 排課推薦」與「AI 課綱比較」共用同一份選填個人資料。使用者可在任一工具中直接填寫，切換工具後立即看到相同內容，不必離開目前工作流程，也不會產生兩份可能不同步的資料。

## 使用者流程

### AI 排課推薦

1. 使用者進入「AI 功能 → AI 排課推薦」。
2. 頁面顯示共用個人資料區，預設展開。
3. 使用者可填寫自我介紹、未來方向、這學期想達成什麼與排課偏好。
4. 送出推薦時，系統讀取這四個欄位並與課表、實習設定一起傳給 AI。

### AI 課綱比較

1. 使用者進入「AI 功能 → AI 課綱比較」。
2. 課程選擇器上方顯示可展開的「個人資料（選填）」區塊，預設收合，避免擠壓 2–5 門課程的選擇流程。
3. 收合狀態顯示填寫完成度，例如「已填 2／4 項」；完全未填時顯示「尚未填寫」。
4. 使用者可直接展開並修改四個欄位，不必跳到排課推薦頁。
5. 執行 AI 比較或產生 ChatGPT 提示詞時，系統讀取同一份共用資料；已填資料會加入個人化取捨，未填仍提供客觀比較。

## 共用資料架構

採用單一 DOM 表單模組，而不是在兩頁各建立一份欄位。DOM 是瀏覽器畫面上的實際介面元素；單一模組代表畫面中始終只有一套真正的輸入欄位。

- 共用模組包含 `ai-profile`、`ai-future`、`ai-goals`、`ai-preferences` 四個既有欄位。
- AI 排課推薦與 AI 課綱比較各提供一個掛載位置。
- 切換 AI 工具時，程式將同一個共用模組移到對應掛載位置。瀏覽器會保留輸入值、游標狀態與欄位事件，因此不需要複製或雙向同步兩份資料。
- AI 請求維持透過既有欄位 ID 取得內容，避免改變後端請求格式。

## 介面設計

### 共用個人資料區

- 標題：`個人資料（選填）`。
- 輔助文字：說明填寫後建議會更貼近個人方向，未填仍可使用。
- 完成度使用文字呈現，不只依靠顏色，確保易讀與無障礙。
- 每個欄位保留可見標籤、範例文字、2000 字上限與至少 44px 的觸控操作高度。
- 延續現有「雨後日光／輕夢核」視覺語言，以淡紫色邊線與柔和日光色點綴，不新增另一套色彩系統。

### 排課推薦狀態

- 共用個人資料區預設展開，因為個人資料是產生推薦方案的主要輸入。
- 主要按鈕與推薦結果維持現有順序。

### 課綱比較狀態

- 共用個人資料區位於工具說明與課程選擇器之間。
- 預設收合；使用者展開後可直接編輯。
- 移除原本「前往填寫」跳轉按鈕，改以區塊本身提供填寫入口。
- 收合與展開使用原生 `details`／`summary` 行為，支援鍵盤操作並保留清楚焦點狀態。

## 隱私與保存

- 個人資料只存在目前頁面的記憶體與輸入欄位中。
- 不寫入 `localStorage`、伺服器、網址、排課 JSON 匯出檔或長期資料庫。
- 重新整理或關閉分頁後清除，符合目前 API Key 與個人敘述的隱私說明。
- AI 功能送出時才會透過使用者自己的 Gemini API Key 傳送；ChatGPT 備援仍只產生提示詞，由使用者自行貼上送出。

## 錯誤與邊界狀態

- 四個欄位皆可留空，不顯示必填錯誤。
- 部分填寫時，只傳送已填內容；完成度以非空白欄位數計算。
- 只有空格的欄位視為未填。
- 切換 AI 工具不得清空資料或重設已展開的輸入內容。
- AI 請求失敗時，共用個人資料與已選比較課程均保留，方便直接重試。

## TDD 驗收標準

依 Red → Green → Refactor 垂直切片逐項完成：

1. 先建立失敗測試，確認共用個人資料模組只有一份且兩個 AI 工具各有掛載位置；完成最小結構後測試轉綠。
2. 先建立失敗測試，確認切換到比較工具時會移動同一個模組並保留欄位內容；完成最小切換邏輯後測試轉綠。
3. 先建立失敗測試，確認完成度會忽略空白並顯示 `已填 N／4 項`；完成最小計算與更新邏輯後測試轉綠。
4. 先建立失敗測試，確認 AI 推薦、Gemini 課綱比較與 ChatGPT 提示詞都讀取相同欄位；完成整合後測試轉綠。
5. 在測試保護下移除舊的「前往填寫」跳轉流程並整理樣式，確保全套測試維持通過。

## 瀏覽器驗證

- 桌機：在排課推薦填入四個欄位，切換至課綱比較後內容完整保留；在比較頁修改後切回推薦亦同步。
- 手機：共用資料區、課程選擇器與比較按鈕不得穿模或產生水平捲動。
- 鍵盤：可用 Tab、Enter 與 Space 操作收合區、輸入欄位與主要按鈕，焦點清楚可見。
- 隱私：重新整理頁面後個人資料清除；匯出的排課 JSON 不包含個人資料。
- 回歸：AI 推薦、AI 課綱比較、ChatGPT 提示詞、候選課程選擇與既有排課功能皆維持正常。

## 不在本次範圍

- 不新增登入帳號或跨裝置同步。
- 不把個人資料寫入瀏覽器長期儲存。
- 不改變 Gemini 或 ChatGPT 的模型與 API Key 流程。
- 不新增更多個人資料欄位。
