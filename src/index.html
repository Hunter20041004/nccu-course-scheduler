第一次開始排課

  1. 先到「匯入／新增」搜尋或匯入你想考慮的課。
  2. 回到「候選課程」檢查時間、資格與課綱，再點課程加入左側課表。
  3. 設定自己的修課條件與實習時間，處理所有衝堂與提醒。
  4. 需要更多協助時，再使用 AI 產生三個推薦方案或匯出手機桌布。

資料保存：新訪客的候選課程與課表會從空白開始，不會看到其他人的內容。同一個網址與同一個瀏覽器會保留你的排課資料；換裝置、換瀏覽器、清除網站資料，或改用另一個正式站網址，都會重新開始。一般排課不需要 API Key，快速導覽也只會切換畫面與指示位置，不會改你的課表。

用三種方式加入候選課程

  1. 政大 115-1 課程庫：在「匯入／新增」輸入課名、教師或九碼課號。新課按「加入候選」；已在候選的課按「更新官方資料」。更新只替換最新官方欄位，會保留你目前的選課、鎖定、上課方式與班別安排。這個方式不需要 API Key。
  2. AI 截圖辨識：上傳政大課程追蹤清單截圖，系統會辨識課名與課號，再核對政大公開資料。這個方式需要自己的 Gemini API Key。
  3. 手動新增：自行輸入課程名稱、學分、星期與時間，也能新增社團、課外組織或個人行程。

沒有開的課不會自動匯入。辨識結果找不到唯一的 115-1 課程，或公開資料不足時，會列在「待確認」，請先核對再決定是否保留。

查看課程時間、完整詳細資料與官方課綱

  1. 候選課程列會直接顯示課程時間與政大節次代號，例如「週四 34」。
  2. 課很多時，展開「篩選」：課程狀態、星期與上課時段可複選；同一類別符合任一項，不同類別要同時符合。「非同步／時間未定」可找沒有固定占用課表的課，按「清除篩選」可取消所有勾選，但不會清掉搜尋文字。
  3. 按「詳細」會在該課程下方展開完整詳細資料；同時間只會展開一門，避免畫面互相遮住。
  4. 完整資料包含課名、課號、授課教師、學分、上課時間、上課形式、修課資格、資料來源、官方備註與班別時段。
  5. 課綱會顯示三種狀態：「可以查看課綱」可直接開啟政大官方頁面；「老師尚未上傳課綱」代表官方查詢成功，但老師目前還沒有提供課綱連結。
  6. 「課綱狀態暫時無法確認」代表這次沒有完成官方查詢，不等於沒有課綱；按「重新查詢官方資料」即可再試。

設定與理解修課條件

  1. 到「選課條件」勾選你符合的系所、雙主修、學程、年級、先修課或其他身分。
  2. 匯入有新限制的課程時，系統會建立對應的可勾選條件，並列出為什麼需要及影響哪些課。
  3. 「日文系擴大輔系課程」這類課程分類只是資訊，不代表只有該身分才能修；真正限制會使用「僅限」、「限修」或其他明確資格文字。
  4. 候選課程顯示「條件不符合，請看詳細。」時，按「詳細」閱讀缺少的條件，再回來更新個人資格。

注意：勾選條件只代表你自行確認符合，不會替代政大選課系統的正式審核。

加入、鎖定、刪除與清空

  1. 點候選課程的主要區塊，可將課程加入左側課表；再點一次可移出。資格不符合或時段尚未選定時，請先處理提示。
  2. 每門候選課右側保留「詳細」；按「•••」可查看課綱、鎖定／解鎖或刪除。鎖定課會留在目前課表，也會在 AI 方案中保留。
  3. 刪除會移除候選課程，也會一併移除課表區塊與鎖定狀態；執行後右下角會提供 15 秒「復原」。
  4. 頁首「更多操作」內的「清空目前課表」只清除已排課程；「清空候選課程」則清除整份候選與相關排課狀態。兩者都可在 15 秒內復原。

實體／同步／非同步與衝堂

  1. 標示可遠距的課程可切換「實體／同步／非同步」，請依你實際打算採用的上課方式選擇。
  2. 選擇非同步後，不會占用每週固定課表時段；但同步活動、考試或一次性活動仍可能產生衝堂。
  3. 有多位老師、班別或兩種官方討論安排的課程，加入後必須先選擇其中一種安排，系統不會把互斥時段同時排入。
  4. 出現衝堂提醒時，請更換官方時段、改用課程允許的非同步形式，或移除其中一門。含衝堂的 AI 方案不會顯示。

安排實習、社團與個人行程

  1. 到「實習設定」輸入目標天數與每天希望保留的開始、結束時間。
  2. 選擇自動模式時，系統會從無衝堂時段找可實習日；選擇固定模式時，可指定每週的固定實習時段。
  3. 實習區塊會畫在課表上，與課程重疊時會出現提醒；摘要會分開顯示「已確認」與因另約課程而「待確認」的天數。
  4. 到「匯入／新增」可加入社團、課外組織、打工、自修或其他個人行程，這些行程也會參與衝堂檢查。

選擇 AI 功能並安全使用自己的 API Key

  1. 前往 Google AI Studio，使用 Google 帳號建立 Gemini API Key；學校帳號受限時請改用個人帳號。
  2. 回到網站按「API Key 未設定」，貼上 Key 並驗證。API Key 等同使用者自己的額度，請不要傳給別人。
  3. 打開「AI 功能」，選擇「AI 排課推薦」或「AI 課綱比較」。兩項工具分開顯示，但會共用你已填寫的目標與偏好。
  4. 使用「AI 排課推薦」時，填寫自我介紹、學期目標與排課偏好，明確寫出最低學分、實習天數、想修的領域、語文偏好或不想考試等需求。
  5. 系統會提供三個推薦方案。先按「預覽」檢查課程、學分、實習與提醒，再按「套用此方案」。鎖定課會保留，非同步課可補學分，但仍須符合資格。
  6. 比較衝堂或內容相近的課：進入「AI 功能」的「AI 課綱比較」，在比較頁搜尋並勾選 2 至 5 門候選課程，再按「AI 比較」。結果會分開顯示確定性衝堂、內容重疊、各課獨有內容、評量、負擔與資料限制。
  7. 選填個人資料：學期目標、未來方向與排課偏好都是選填；未填仍可取得客觀比較，填寫後才會加入更精準的個人化取捨建議。
  8. 不用網站內 AI：可按「帶到 ChatGPT」。網站會準備並嘗試複製提示詞，再開啟 ChatGPT；不會替你自動送出訊息，你可以先檢查內容再貼上送出。

匯出手機桌布與常見問題

  1. 手機預設使用「行程」模式依星期與時間排列；要核對政大節次時切到「方格」。排完後可匯出包含非同步、時間未定與提醒的手機桌布。
  2. AI 顯示未設定：目前分頁沒有有效 Key，請重新貼上；重新整理後也必須重新設定。
  3. 課程待確認：代表無法從公開資料找到唯一結果或資料不足，請用課號與官方課綱核對。
  4. 官方資料更新：重新查詢失敗時，原有候選課程與排課資料會保留，請稍後再按「重新查詢官方資料」。更新成功後，重新整理網頁仍會保留新的官方資料。
  5. 換手機或網址沒有資料：資料只存在原本瀏覽器。從頁首「更多操作」匯出排課 JSON，在另一台裝置用「從檔案匯入」預覽並確認；檔案不含 Key、AI 自我介紹或截圖。
  6. 清除錯誤內容：在「更多操作」清空課表或候選清單；誤按時立刻使用 15 秒復原。
