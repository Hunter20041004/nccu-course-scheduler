第一次使用

新訪客的候選課程與課表會從空白開始，不會看到其他人的內容。同一個網址與同一個瀏覽器會保留你的排課資料；換裝置、換瀏覽器或清除網站資料則會重新開始。不設定 API Key 也能搜尋官方課程、排課、管理條件與設定實習。快速導覽只會切換畫面與指示位置，不會改你的課表。

取得 Gemini API Key

  1. 前往 Google AI Studio。
  2. 登入 Google 帳號；學校帳號受限時請改用個人帳號。
  3. 建立 API Key。
  4. 複製 Key 後回到網站貼上。

API Key 等同使用者自己的額度，請不要傳給別人。Key 不會存到伺服器、資料庫或網址；關閉或重新整理分頁即清除。GitHub Pages 分享版可測試一般排課與桌布匯出；AI 匯入與推薦請改用完整 Live Demo。

匯入候選課程

  1. 政大 115-1 課程庫：輸入課名、教師或九碼課號直接搜尋，再按「加入候選」；不需要 API Key。
  2. AI 截圖辨識：上傳政大課程追蹤清單截圖，系統會辨識並核對公開課程資料；需要自己的 Gemini API Key。
  3. 手動新增：自行輸入課程、社團、課外組織、打工或私人行程。

沒有開的課不會自動匯入；資料不足或資格不明的課程會標為待確認，請再查看詳細資訊。

管理選課條件

有系所、雙主修、學程、年級或身分限制的課會出現在選課條件。條件不符合時請看詳細。

排課操作

點候選課程可加入課表；詳細、鎖定、刪除可管理每門課。任何課程都能鎖定，鎖定課會在 AI 方案中保留。

實習與個人行程

可設定實習天數、固定時段、彈性時段，也可新增社團、課外組織、打工或自修來檢查衝堂。

AI 推薦方案

寫清楚最低學分、想留幾天實習、主題偏好、語文課偏好和不想考試等條件。有衝堂的方案不應顯示；非同步課可以補學分但仍需符合資格。

常見問題

如果 AI 說 API Key 沒設定，代表目前分頁還沒貼 Key。若截圖匯入後有待確認，請自行核對政大課程資料。想重跑方案時，回到 AI 推薦重新送出需求即可。
