第一次開始排課

  1. 先到「匯入／新增」搜尋或匯入你想考慮的課。
  2. 回到「候選課程」檢查時間、資格與課綱，再點課程加入左側課表。
  3. 設定自己的修課條件與實習時間，處理所有衝堂與提醒。
  4. 需要更多協助時，再使用 AI 產生三個推薦方案或匯出手機桌布。

資料保存：新訪客的候選課程與課表會從空白開始，不會看到其他人的內容。同一個網址與同一個瀏覽器會保留你的排課資料；換裝置、換瀏覽器、清除網站資料，或改用另一個正式站網址，都會重新開始。一般排課不需要 API Key，快速導覽也只會切換畫面與指示位置，不會改你的課表。

用三種方式加入候選課程

  1. 政大 115-1 課程庫：在「匯入／新增」輸入課名、教師或九碼課號，按「搜尋」後選擇「加入候選」。這個方式不需要 API Key。
  2. AI 截圖辨識：上傳政大課程追蹤清單截圖，系統會辨識課名與課號，再核對政大公開資料。這個方式需要自己的 Gemini API Key。
  3. 手動新增：自行輸入課程名稱、學分、星期與時間，也能新增社團、課外組織或個人行程。

沒有開的課不會自動匯入。辨識結果找不到唯一的 115-1 課程，或公開資料不足時，會列在「待確認」，請先核對再決定是否保留。

查看課程時間、完整詳細資料與官方課綱

  1. 候選課程列會直接顯示課程時間與政大節次代號，例如「週四 34」。
  2. 按「詳細」會在該課程下方展開完整詳細資料；同時間只會展開一門，避免畫面互相遮住。
  3. 完整資料包含課名、課號、授課教師、學分、上課時間、上課形式、修課資格、資料來源、官方備註與班別時段。
  4. 按「課綱」或詳細區底部的「開啟官方課綱」，可前往政大官方課綱。若公開資料沒有安全連結，系統會明確標示無法開啟。

設定與理解修課條件

  1. 到「選課條件」勾選你符合的系所、雙主修、學程、年級、先修課或其他身分。
  2. 匯入有新限制的課程時，系統會建立對應的可勾選條件，並列出為什麼需要及影響哪些課。
  3. 「日文系擴大輔系課程」這類課程分類只是資訊，不代表只有該身分才能修；真正限制會使用「僅限」、「限修」或其他明確資格文字。
  4. 候選課程顯示「條件不符合，請看詳細。」時，按「詳細」閱讀缺少的條件，再回來更新個人資格。

注意：勾選條件只代表你自行確認符合，不會替代政大選課系統的正式審核。

加入、鎖定、刪除與清空

  1. 點候選課程的主要區塊，可將課程加入左側課表；再點一次可移出。資格不符合或時段尚未選定時，請先處理提示。
  2. 按「鎖定」可鎖定任何課程，不限必修。鎖定課會留在目前課表，也會在 AI 方案中保留。
  3. 按「刪除」會移除該候選課程，也會一併移除它在課表中的區塊與鎖定狀態。
  4. 「清空目前課表」只清除已排課程，候選清單仍在；「清空候選課程」會清除整份候選清單與相關排課狀態。

實體／同步／非同步與衝堂

  1. 標示可遠距的課程可切換「實體／同步／非同步」，請依你實際打算採用的上課方式選擇。
  2. 選擇非同步後，不會占用每週固定課表時段；但同步活動、考試或一次性活動仍可能產生衝堂。
  3. 有多位老師、班別或兩種官方討論安排的課程，加入後必須先選擇其中一種安排，系統不會把互斥時段同時排入。
  4. 出現衝堂提醒時，請更換官方時段、改用課程允許的非同步形式，或移除其中一門。含衝堂的 AI 方案不會顯示。

安排實習、社團與個人行程

  1. 到「實習設定」輸入目標天數與每天希望保留的開始、結束時間。
  2. 選擇自動模式時，系統會從無衝堂時段找可實習日；選擇固定模式時，可指定每週的固定實習時段。
  3. 實習區塊會畫在課表上，與課程重疊時會出現排課提醒；半天與整天會依實際可用時數計算。
  4. 到「匯入／新增」可加入社團、課外組織、打工、自修或其他個人行程，這些行程也會參與衝堂檢查。

取得 API Key 並使用 AI 推薦

  1. 前往 Google AI Studio，使用 Google 帳號建立 Gemini API Key；學校帳號受限時請改用個人帳號。
  2. 回到網站按「API Key 未設定」，貼上 Key 並驗證。API Key 等同使用者自己的額度，請不要傳給別人。
  3. 在「AI 推薦」填寫自我介紹、學期目標與排課偏好，明確寫出最低學分、實習天數、想修的領域、語文偏好或不想考試等需求。
  4. 系統會提供三個推薦方案。先按「預覽」檢查課程、學分、實習與提醒，再按「套用此方案」。鎖定課會保留，非同步課可補學分，但仍須符合資格。

Key 安全：Key 只保留在目前分頁記憶體，不會寫入伺服器、資料庫、網址或瀏覽器長期儲存；關閉或重新整理分頁即清除。GitHub Pages 分享版可使用一般排課與桌布匯出，AI 功能請使用完整 Live Demo。

匯出手機桌布與常見問題

  1. 排完後按課表上方的「匯出手機桌布」，下載包含課表、非同步與時間未定課程及排課提醒的直式圖片。
  2. AI 顯示未設定：目前分頁沒有有效 Key，請重新貼上；重新整理後也必須重新設定。
  3. 課程待確認：代表無法從公開資料找到唯一結果或資料不足，請用課號與官方課綱核對。
  4. 換手機沒有資料：資料只存在原本瀏覽器與網址，不會跨裝置或跨網站同步。
  5. 清除錯誤內容：使用「清空目前課表」保留候選課，或使用「清空候選課程」重新建立清單。
